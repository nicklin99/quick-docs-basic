## 关于极速文档主题

极速文档英文命名为Quick Docs Baisc，是基于Hugo制作的一个文档类型基础主题，快速搭建技术文档，快速显示给用户。

本主题不会生成静态文件，也不是一个网站，请使用Hugo快速搭建站点配套使用。
